# Como acessar o modelo C4

1. Abra o site do [Structurizr Playground](https://playground.structurizr.com/)
2. Copie o conteúdo do arquivo `docs/model/model.c4`
3. Cole no dashboard de código do site aberto e execute

Para o escopo do projeto, foi apenas definido até o nível 3 do modelo. É possível acessar os três níveis por meio da aba na parte superior do diagrama gerado, que contém as abas:

* System Context View: Nossa solução (#C1)
* Container View: Nossa solução (#C2)
* Component View: Nossa solução - Back-end (#C3)
